# Three.js Gradient Mesh

This is an experiment using `three.js` built-in meshes and light settings with `lil-gui` debug tools.

## Preview

<img src="./app.png" alt="Gradient Mesh App" />

## Installation and Usage

```bash
pnpm install
pnpm dev
```
